Setting Up Git for VSCode

Step 1

Make sure you are logged in to your Github account. 🔐

Step 2

Open Dev Challenge Repository 👈

Step 3

Click on Fork Button. 🍴

Step 4

You should now see this repository in your profile. 👀

Step 5

Copy the repository. 👥

Step 6

Open VSCode, Click clone git repo, paste the link we copied in previous step & hit Enter. 🖱️

Step 7

Save this repo anyhwere you want. I'll save it on Desktop. 💾

Step 8

Open the cloned repository. 👥 ⬅️

Step 9

Now you'll be in main branch. Click on bottom left icon to change the branch. I'll choose c1 which is Challenge 1. 🌴

That's It.

Now you're ready to solve the challenge. 😀
